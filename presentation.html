What is preferential data?

  • One entry = a ranking among a common set of items.

  • e.g., ballots for the house of reps!

  • ‚Äúcomputational social choice, recommender systems, data mining, machine learning, combinatorial optimization, to name just a few‚Äù ‚Äî PrefLib

  • Other kinds of preferences can be incomplete or even include ties

nsw_election_files <- list.files("nswla_data/", pattern = "\\.zip$", full.names = TRUE)

# Load the entire election with readr + dplyr
election <- lapply(
  nsw_election_files,
  \(x) read_delim(x, delim = "\t", show_col_types = FALSE) |>
         select(District, PollingPlaceName, BPNumber, CandidateName, PrefCounted)
) |>
  bind_rows() |>
  print(n = 4)
